Global Incident Map API

A hosted REST API ready to respond to your requests for Global Incident Map disease reports.

This API allows you to access various disease reports available upon your request based on your input of:

  • Time period of interest (Start Time and End Time)
  • Key terms
  • Location

Get Started - Example Use Cases

GET /Articles/

The articles details endpoint will be the main endpoint that users call. It will return a large json string containing a list of articles. The return format is specified below.

Output Format


$ curl -X GET "http://127.0.0.1:5100/articles/?start%20time=2019-02-17T22%3A41%3A00&end%20time=2019-02-18T22%3A41%3A00&keyterms=Anthrax%2CZika&location=Sydney" -H "accept: application/json"

[

{

"url": "newspaper.com/thisarticle",

"date_of_publication": "<string::date>",

"headline": "...",

"main_text": "<string>",

"reports": [...]

}

]
